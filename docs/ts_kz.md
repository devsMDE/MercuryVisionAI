# MercuryVision Studio 2.1 – Техникалық сипаттама (ТС)

## 1. Кіріспе
### 1.1 Мақсаты
Бұл құжат MercuryVision Studio жобасының толық Техникалық сипаттамасы болып табылады. Ол геокеңістіктік бейнелерді талдауға және өзгерістерді анықтауға арналған жасанды интеллект негізіндегі премиум веб-платформа.

### 1.2 Қолдану аясы
Бұл құжат бағдарламалық архитектураны, «Apple» деңгейіндегі UI/UX дизайн стандарттарын, жоғары деңгейлі қауіпсіздік шараларын және PHP, Firebase, сондай-ақ ChangeFormer модельдерін пайдаланатын серверлік интеграцияларды сипаттайды.

## 2. Архитектура
### 2.1 Фронтенд 
- **Технологиялар:** HTML5, Vanilla JavaScript (ES6+), CSS3 айнымалылары.
- **Пайдаланушы интерфейсі (UI):** Глассморфизм стилі, бейімделгіш орналасу (адаптивті) және микро-анимациялар.
- **Иконика:** Lucide белгішелері.

### 2.2 Бэкенд (Сервер)
- **Негізгі тіл:** API сұраныстарын өңдейтін PHP 8+.
- **Дерекқор (Database):** JSON форматындағы жергілікті дерекқор (`api/json_db.php`), PDO интерфейсіне имитация жасайды және Firebase Realtime Database-пен қатар жұмыс істейді.
- **Аутентификация:** Google Firebase Authentication жүйесі (JWT).

## 3. Негізгі функциялар
- **AI Салыстыру (Compare):** Жалпы бейнелер арасындағы өзгерістерді анықтау.
- **Арнайы режимдер:** Су динамикасы, Орман аналитикасы, NDVI / Ауыл шаруашылығы индексі, Қала кеңістігін талдау.
- **Стратегиялық AI-Аналитик (Чат):** Талдау нәтижелері мен метрикаларға сүйене отырып, мәнмәтінді сақтайтын және қорытынды шығаратын жасанды интеллект көмекшісі.

## 4. Дизайн (UI/UX) талаптары
- **Тегіс анимациялар (Fluid Micro-Animations):** Барлық интерактивті элементтер `hover`, `focus` немесе `click` кезінде `cubic-bezier` тайминг функциясы арқылы жұмсақ ауысуы міндетті.
- **Премиум эстетика:** Жайпақ (жазық) түстерден аулақ болу. Терең көлеңкелер (`box-shadow`) мен күрделі градиенттерді қолдану.

## 5. Қауіпсіздік талаптары
- **Деректерді қорғау:** API кіріс-шығыс деректері XSS және SQL-инъекцияларының алдын алу үшін қатаң түрде сүзгіден (валидациядан) өтуі тиіс.
- **CORS саясаты:** `config.php` файлында рұқсат етілген домендер мен 헤дерлерге шектеу қою.
- **Құпияларды сақтау:** Пайдаланушылардың жеке AI API кілттері (OpenAI/Gemini) браузердің `localStorage` жадысында ғана сақталады және ашық түрде тасымалданбайды.

## 6. Мақсатты аудитория
Суреттерді (жер серігі немесе топографиялық) «дейін-кейін» форматында зерттейтін, жоғары дәлдікті талап ететін сарапшылар, зерттеушілер және ірі кәсіпорындардың шешім қабылдаушы тұлғалары.
